Solidarity: Comprehensive App for Learning UBCV

Android|Java

Solidarity is a comprehensive Android app for the University of Batangas' Core Values Subjects. It's an E-Learning app consisting the Digital Copy of the UBCV books. This app gives trivia questions at every end of the lesson and Chapter Quizzes at every end of the chapter. I used Java as it's development language and Firebase for Authentication and Database.

GoTalk: Community Chat App

Android|Kotlin

GoTalk is a community chat app for Android. Users can register, log-in and create a topic that will be visible to everyone. Users can enter and chat all people within the group. This app was made using Kotlin as its programming language, a custom API made by DevSlopes for broadcast and authentication and MongoDB for the database.

Boxes, Pies and Knives

Unity Android|C#

Boxes, Pies and Knives is a 2.5D Object-Dodger Platformer Game made with Unity. The mechanics in this game is very simple; Be a box, dodge the falling knives, catch the pies and survive as long as you can, Whoever stays alive the longest, wins.
